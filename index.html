Anleitung für neue Nutzer

1. Sortenberater öffnen

145 Arche Noah Sorten sind bereits verfügbar. Lass dich beraten, welche Sorten für deinen Standort passen.

  • Standort-Berater: Wähle Höhenlage, Exposition und Spätfrostrisiko
  • Sortenübersicht: Durchsuche alle Sorten nach Quelle und Verwendung

2. Baumkataster anlegen

Trage deine Bäume ein oder importiere eine Excel-Datei.

  • Eigene IDs können beim Import und Anlegen verwendetet werden
  • Bäume können später im Lageplan positioniert werden

3. Lageplan erstellen

Markiere deine Bäume auf dem Luftbild oder nutze die Satellitenkarte.

  • Wechsel zwischen Luftbild und Satellit
  • Bäume per Klick setzen (auf beiden Karten)
  • GPS-Kalibrierung für Genauigkeit (optional)

4. Sorten importieren

Excel oder CSV Datei hochladen und Spalten automatisch zuordnen.

  • Unterstützte Formate: .xlsx, .xls, .csv, .tsv
  • Drag & Drop oder Klick zum Hochladen
  • Spalten werden automatisch erkannt
  • Eigene IDs werden beibehalten
  • Als Vorlage speichern für spätere Imports
  • Re-Import: Gleiche Datei ergänzt vorhandene Daten

5. Import verwalten

Alle Imports werden protokolliert und können rückgängig gemacht werden.

  • Import-Details: Klick auf Import-Eintrag zeigt Details
  • Undo: Vor jedem Import wird ein Snapshot erstellt (max. 5)
  • Löschen: Importierte Bäume können entfernt werden

Pflückzeit & Erntevorhersage

Jede Sorte hat einen Original-Pflückzeitpunkt (z.B. 10,5 = Mitte Oktober). Aus deinen Ernte-Einträgen der letzten 3 Jahre berechnet die App einen Mittelwert (z.B. 9,8 = Anfang Oktober).

Original-Pflückzeit
Der historische Referenzwert aus der Sortenbeschreibung. Bleibt unverändert.
Mittelwert (→ 9,8)
Berechnet aus deinen tatsächlichen Ernte-Dokumentationen. Zeigt, wann die Sorte bei dir am Hof tatsächlich reif wird.
Erntevorhersage (📊)
Wenn du das Phänologie-Ereignis "Schwarzer Holunder – Blühbeginn" einträgst, wird eine Vorhersage für das aktuelle Jahr berechnet. Der Holunder-Blühbeginn verschiebt sich durch den Klimawandel – und damit auch deine Ernte.

So funktioniert's:

  1. Trage Ernten mit Datum ein (Baum öffnen → Ernte-Dokumentation)
  2. Ab 1 Ernte wird der Mittelwert berechnet
  3. Optional: Trage den Holunder-Blühbeginn ein (Baum öffnen → Phänologie)
  4. Die Vorhersage zeigt: Original → Mittelwert → 📊 Vorhersage

Quelle Phänologie: Naturschutzmonitoring.at – Phänoflex (ÖPUL-Programm)

FAQ

Wie installiere ich die App?
URL aufrufen → "Zum Home-Bildschirm" (iOS: Safari Teilen → Android: Chrome Installieren)
Funktioniert die App offline?
Ja, nach dem ersten Laden.
Wie sichere ich meine Daten?
"Daten & Sicherung" → "Exportieren"
Kann ich meine Daten auf ein anderes Gerät übertragen?
Ja, als JSON exportieren und importieren.
Wie ändere ich meine eigene ID?
Baum bearbeiten → ID-Feld ändern.
Was passiert beim Re-Import?
Bei erneutem Import der gleichen Datei werden nur leere Felder ausgefüllt. Vorhandene Daten bleiben erhalten.
Kann ich einen Import rückgängig machen?
Ja, im Import-Management kannst du "Undo" klicken. Es werden max. 5 Snapshots gespeichert.
